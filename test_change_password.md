# 更改密碼功能測試指南

## 測試步驟：

1. **開啟瀏覽器**
   - 訪問 http://localhost:3000

2. **使用學生帳號登入**
   - 帳號：11430001（或其他學生帳號）
   - 密碼：fhsh1234（或對應的密碼）

3. **點選「更改密碼」按鈕**
   - 在頁面右上角，「登出」按鈕旁邊應該會看到藍色的「更改密碼」按鈕

4. **測試更改密碼Modal**
   - 輸入舊密碼：fhsh1234
   - 輸入新密碼：test1234
   - 輸入確認新密碼：test1234
   - 點選「確認更改」

5. **驗證更改成功**
   - 應該看到「密碼更改成功！」的提示
   - Modal 自動關閉

6. **登出並使用新密碼登入**
   - 點選「登出」
   - 使用新密碼 test1234 登入
   - 確認可以成功登入

7. **改回原密碼（可選）**
   - 登入後再次更改密碼
   - 舊密碼：test1234
   - 新密碼：fhsh1234
   - 確認密碼：fhsh1234

## 預期結果：

✅ 學生登入後可以看到「更改密碼」按鈕
✅ 點選按鈕後出現更改密碼Modal
✅ 驗證功能正常：
   - 空欄位會提示錯誤
   - 新密碼與確認密碼不一致會提示錯誤
   - 新密碼長度小於6會提示錯誤
   - 新密碼與舊密碼相同會提示錯誤
   - 舊密碼錯誤會提示錯誤
✅ 更改成功後可以使用新密碼登入
✅ 教師端不顯示「更改密碼」按鈕
